Course Identification


Les exigences conceptuelles
 application client-Serveur multicouches (les couches
Gui, Business et Data).

- Couche GUI :
Application Site Web C # avec les formulaires web, html, CSS en fonction des
exigences de fonctionnalité de l’utilisateur.

- Couche DATA :
Classe qui encapsule la base de données (accès) et fournit des interfaces publiques
pour alimenter les données dans la couche métier et l'écriture des données dans la
base de données.
Base de données Microsoft Access qui contiendra toutes les données nécessaires pour
Remax.


Les exigences techniques et fonctionnelles
• Pour la couche GUI : Utiliser des webformes avec interfaces user-friendly et
thématiques (couleurs et images reliées a Remax).


• Pour la couche Business :   Concepts de l’Orientée Objet comme la
classification (héritage, composition and agrégation), encapsulation (les objets
dotNet).
• Pour la couche Data :  Ado.net.
